### Minimaalinen electron-react -sovellus

```bash
git clone https://github.com/toppyy/react-electron-boilerplate
cd react-electron-boilerplate
npm install
npm run start
```

Kehitysympäristö:
```bash
git clone https://github.com/toppyy/react-electron-boilerplate
cd react-electron-boilerplate
npm install

npm run dev
```
Ja toisessa terminaalissa: 
```bash
npm run start-dev
```
